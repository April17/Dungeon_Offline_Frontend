Thank you for purchasing Fantasy Top-down Game Kit. With this pack you will create an awesome Fantasy Top-down game. With everything you need to get started, tiles, effects and characters made by a professional pixel artist. + some hud parts and touch controls

Files

Start by extracting your zip-file downloaded from Envato with any extraction software to a local folder. For MAC/OSX you can use UnrarX and for PC/Windows you can go with WinRAR.

Folders

All images are located in the folders where you extracted the zip-file, they are well structured to help you get started quicker. The images are all in transperant PNG and can be edited with any image software.

 

Animation

The animations have all been sliced into different images. Filenames showing what type of animation it is. 

Remember to use a lot of Effects. :)
